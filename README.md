# Commute Podder

This is a web app that (will) allow you to choose a route within stockholm using public transport, and based on the length of the trip it will recommend a good podcast to listen to.

Link to the currently deployed website: 

# https://commutepodderiprog.web.app/

## How does it work?

We are using two APIs, SL api and listenotes api. 

## Architecture

We are using react, vite, tailwind for the front end, 

For the backend we are using typescript, npm and hosting on firebase.
